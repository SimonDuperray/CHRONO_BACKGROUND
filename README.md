# CHRONO_BACKGROUND
CHRONO_BACKGROUND is a simple web page standing the remaining seconds before the end of the day. It can help some people to be more producing.
## Installation
```bash
Put index.html and rmnchr.bat in "C:\Windows\System32\"
```

## Usage
```text
in terminal: rmnchr
```
